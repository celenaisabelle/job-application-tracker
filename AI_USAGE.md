## Tools Used
- ChatGPT for assistance with HTML file structure, templates, app routes, and algorithms. 

## Key Prompts
- "Help create the directory and file structure of the project."
- "Assist me with the job match algorithm"
- "Let me know what I can adjust with the database to improve job match results"

## What Worked Well
- The AI provided helpful guidance when setting up the project structure
- Assistance with the job match algorithm made it easier to implement and debug the matching logic
- Suggestions on database adjustments improved the accuracy of job match results

## What I Modified
- Updated the project structure to ensure files and routes were properly organized
- Refined the job match algorithm to compare user skills with job descriptions
- Adjusted the database content (job descriptions) to improve matching accuracy

## Lessons Learned
- Always verify AI's responses are accurate.  
- Do not create dependency with AI. Only use it as a learning tool.
